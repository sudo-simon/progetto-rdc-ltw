## **Setup**

Per inizializzare l'ambiente Docker e il database, dalla root del progetto eseguire:

1. `~$ sudo docker-compose up`
2. `~$ ./initDB.sh`
   
Poi visitare:

3. **[Sapiens](https://localhost:8887)** (localhost:8887)
